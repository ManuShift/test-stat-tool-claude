Ziel: Verschiedene KI-Bots (GPTBot, ClaudeBot, PerplexityBot …) hinterlassen charakteristische Spuren im Apache Access-Log. Diese Umgebung erlaubt den direkten Vergleich von GoAccess, AWStats und einem eigenen Python-Analyzer.

Apache-Log liegt unter ./logs/access.log — direkt für alle Tools zugänglich.
